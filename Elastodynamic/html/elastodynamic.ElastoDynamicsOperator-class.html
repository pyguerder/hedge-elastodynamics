An nD linear Elastodynamics operator.

dq/dt - dF/dx - dG/dy - dH/dz = 0

where e.g. in 3D

q = (rho_v_1, rho_v_2, rho_v_3, F_11, F_22, F_33, 2*F_23, 2*F_13, 2*F_12) F = (P_11, P_12, P_13, v_1, 0, 0, 0, v_3, v_2) G = (P_12, P_22, P_23, 0, v_2, 0, v_3, 0, v_1) H = (P_13, P_23, P_33, 0, 0, v_3, v_2, v_1, 0)

For the linear case with included attenuation the stress-strain relation is given by

P_ij = C_ijkl * F_kl + nu_ijkl * dv_k/dx_l

Field order is [rho_v_1, rho_v_2, rho_v_3, F_11, F_22, F_33, 2*F_23, 2*F_13, 2*F_12].
